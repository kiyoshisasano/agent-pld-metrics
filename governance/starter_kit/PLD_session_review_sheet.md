## PLD Session Review Sheet (Lightweight)

**How to use:**

1. Use this sheet to review 5–10 sessions.
2. Fill one table row per session.
3. Discuss results in the first joint review.

---

### Review Table

| Session ID | Drift Occurred? | Repair Type        | Reentry Confirmed? | Outcome                  | Notes       |
| ---------- | --------------- | ------------------ | ------------------ | ------------------------ | ----------- |
| example_01 | Yes / No        | Soft / Hard / None | Yes / No           | Success / Partial / Fail | <1–2 lines> |
| example_02 | Yes / No        | Soft / Hard / None | Yes / No           | Success / Partial / Fail | <1–2 lines> |

---

### Quick Reference (Plain Language)

* **Drift =** The system went off track.
* **Repair =** The system tried to fix the issue.
* **Soft Repair =** Clarification / adjustment.
* **Hard Repair =** Reset / block / restart.
* **Reentry =** Confirmation before moving forward.
* **Outcome =** How the session ended.

Use these terms even if you don’t know PLD’s technical specs.
